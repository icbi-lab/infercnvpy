infercnvpy.tl.umap¶

infercnvpy.tl.umap(adata, neighbors_key='cnv_neighbors', key_added='cnv_umap', inplace=True, **kwargs)¶

Compute the UMAP on the result of infercnvpy.tl.infercnv().

Thin wrapper around scanpy.tl.umap()

Parameters
adata : AnnDataAnnData

annotated data matrix

neighbors_key : strstr (default: 'cnv_neighbors')

Key under which the result of infercnvpy.pp.neighbors() is stored in adata

key_added : strstr (default: 'cnv_umap')

Key under which the result UMAP will be stored in adata.obsm

inplace : boolbool (default: True)

If True, store the result in adata.obsm, otherwise return the result of UMAP.

**kwargs

Additional arguments passed to scanpy.tl.umap().
