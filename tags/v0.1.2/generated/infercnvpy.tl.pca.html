infercnvpy.tl.pca¶

infercnvpy.tl.pca(adata, svd_solver='arpack', zero_center=False, inplace=True, use_rep='cnv', key_added='cnv_pca', **kwargs)¶

Compute the PCA on the result of infercnvpy.tl.infercnv().

Thin wrapper around scanpy.tl.pca().

Parameters
adata : AnnDataAnnData

annotated data matrix

svd_solver : strstr (default: 'arpack')

See scanpy.tl.pca().

zero_center : boolbool (default: False)

See scanpy.tl.pca().

inplace : boolbool (default: True)

If True, store the result in adata.obsm. Otherwise return the PCA matrix.

use_rep : strstr (default: 'cnv')

Key under which the result of infercnv is stored in adata

key_added : strstr (default: 'cnv_pca')

Key under which the result will be stored in adata.obsm if inplace=True.

**kwargs

Additional arguments passed to scanpy.tl.pca().

Return type

ndarray | NoneOptional[ndarray]
